**Files:**

- [ ] **Step 1: Cập nhật layout.tsx để nạp font Inter**

Thay thế font `Geist` bằng `Inter` có subset `vietnamese`:

```typescript
import type { Metadata } from "next";
import { Inter, Geist_Mono } from "next/font/google";
import "./globals.css";

const inter = Inter({
  variable: "--font-sans",
  subsets: ["latin", "vietnamese"],
});

const geistMono = Geist_Mono({
  variable: "--font-geist-mono",
  subsets: ["latin"],
});

  description: "Generated by create next app",
};

export default function RootLayout({
  children,
  ...
```

- [ ] **Step 2: Cập nhật globals.css áp dụng font**

Cập nhật `--font-sans` và gán thuộc tính `font-family` cho body:

```css
@import "tailwindcss";

:root {
  --background: #ffffff;
  --foreground: #171717;
}

@theme inline {
  --color-background: var(--background);
  --color-foreground: var(--foreground);
  --font-sans: var(--font-sans);
  --font-mono: var(--font-geist-mono);
}

body {
  background: var(--background);
  color: var(--foreground);
  font-family: var(--font-sans), Arial, Helvetica, sans-serif;
}
```

- [ ] **Step 3: Commit**

```bash
git add src/app/layout.tsx src/app/globals.css
git commit -m "style: configure Inter font with vietnamese subset globally"
```

### Task 2: Chuyển đổi Board Page sang giao diện sáng Pastel & Thêm Đăng xuất

**Files:**

- [ ] **Step 1: Khai báo hàm `handleSignOut`**

Thêm hàm `handleSignOut` trong `src/app/board/[id]/page.tsx`:

```typescript
  const handleSignOut = async () => {
    try {
      await supabase.auth.signOut();
```
